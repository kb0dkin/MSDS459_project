Guitar Recommender

Welcome to the Guitar Recommender, a final project for the 2023 Winter Quarter of MSDS 432.

This program contains records of thousands of guitars scraped from Guitar Center, an online store for guitars and other musical instruments. If we were to extend this project in the future, we would likely scrape from several more websites to expand our data even further.

This programs requires a local instance of EdgeDB to be running. For more information on that process, please view the ReadMe that is available in the base code directory or on the github repository
